Lekcja 8: id i class

Wróć na główną!

Teoria

Atrybuty <id> i <class> służą do identyfikowania elementów HTML, aby można było je stylować w CSS lub używać w JavaScript.

Atrybut <id> jest unikalny, co oznacza, że może wystąpić tylko raz na jednej stronie. Służy do identyfikowania pojedynczego elementu, np. konkretnego nagłówka lub sekcji.

Atrybut <class> może być używany wielokrotnie i służy do grupowania elementów o wspólnym stylu. Dzięki temu można nadać jeden styl wielu elementom jednocześnie.

W CSS id zapisuje się za pomocą znaku #, np. #naglowek, a class za pomocą kropki, np. .przycisk.

Id stosuje się wtedy, gdy element jest unikalny, natomiast class wtedy, gdy wiele elementów ma wyglądać tak samo.

Przykład

To jest nagłówek z ID

Ten akapit należy do klasy "tekst".

Ten też należy do tej samej klasy.

Ten też należy do innej klasy.

<!-- ID + CSS -->
<h2 id="tytul">To jest nagłówek z ID</h2>

<!-- CLASS + CSS -->
<p class="tekst">Ten akapit należy do klasy "tekst".</p>
<p class="tekst">Ten też należy do tej samej klasy.</p>
<p class="tekst2">Ten też należy do innej klasy.</p>


<button class="przycisk">Kliknij mnie</button>

<!-- CSS -->
<style>
#tytul {
  color: #1a5fb4;
  text-align: center;
}

.tekst {
  color: red;
  font-size: 16px;
}
.tekst2{
      color: green;
      font-size: 30px;
    }

.przycisk {
  background: #1a5fb4;
  color: white;
  padding: 10px 14px;
  border: none;
  border-radius: 6px;
}

.przycisk:hover {
  background: #15467e;
}
</style>
