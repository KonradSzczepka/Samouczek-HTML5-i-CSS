Lekcja 9: CSS podstawy

Wróć na główną!

Teoria

CSS (Cascading Style Sheets) to język stylów używany do opisywania wyglądu stron internetowych. Dzięki CSS można zmieniać wygląd elementów HTML, takich jak kolory, czcionki, rozmiary tekstu oraz układ strony.

CSS nie odpowiada za strukturę strony (to robi HTML), tylko za jej wygląd. Dzięki temu kod jest podzielony na część logiczną (HTML) i wizualną (CSS).

Style CSS można dodawać na trzy sposoby: wewnętrznie w znaczniku <style> w pliku HTML, zewnętrznie w osobnym pliku .css oraz lokalnie bezpośrednio w znaczniku HTML za pomocą atrybutu style.

Najczęściej używa się plików zewnętrznych CSS, ponieważ pozwalają one łatwo zarządzać wyglądem wielu stron jednocześnie.

CSS działa na zasadzie reguł. Każda reguła składa się z selektora (elementu HTML) oraz deklaracji stylu, czyli właściwości i jej wartości.

Przykład

To jest tekst stylowany za pomocą CSS.

Ten tekst ma inny kolor. 

<!-- CSS w atrybucie style (inline) -->

<p style="color: blue; font-size: 20px;">
  To jest tekst stylowany za pomocą CSS.
</p>

<p style="color: green;">
  Ten tekst ma inny kolor.
</p>
