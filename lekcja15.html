Lekcja 15: Menu

Wróć na główną!

Teoria

Menu w stronach internetowych tworzy się z nieuporządkowanej listy <ul> zawierającej elementy <li>, w których znajdują się odsyłacze <a>.

Jest to standardowy sposób budowania menu, ponieważ lista naturalnie grupuje elementy, które są ze sobą powiązane.

Każdy element menu to osobny link, który prowadzi do innej podstrony lub sekcji strony.

Domyślnie lista ma punkty (kropki), dlatego usuwa się je za pomocą list-style: none, aby menu wyglądało jak pasek nawigacyjny.

Układ menu można zmieniać za pomocą CSS, np. ustawiając elementy w jednej linii (menu poziome) lub jeden pod drugim (menu pionowe).

Przykład

✔ MENU POZIOME

  • Start
  • O mnie
  • Kurs
  • Kontakt

✔ MENU PIONOWE

  • Start
  • O mnie
  • Kurs
  • Kontakt
<!-- MENU = NIEUPORZĄDKOWANA LISTA ODSYŁACZY -->

<nav class="menu">
  <ul>
    <li><a href="#">Start</a></li>
    <li><a href="#">O mnie</a></li>
    <li><a href="#">Kurs</a></li>
    <li><a href="#">Kontakt</a></li>
  </ul>
</nav>

<!-- CSS -->

<style>
/* usunięcie kropek */
.menu ul {
  list-style: none;
  margin: 0;
  padding: 0;
}

/* linki */
.menu a {
  text-decoration: none;
  display: block;
  padding: 10px 14px;
  background: #1a5fb4;
  color: white;
}

/* poziome menu */
.menu-horizontal ul {
  display: flex;
  gap: 10px;
}

/* pionowe menu */
.menu-vertical ul {
  width: 200px;
}

.menu a:hover {
  background: #15467e;
}
</style>
