Lekcja 10: Selektory CSS

Wróć na główną!

Teoria

Selektory CSS służą do wskazywania elementów HTML, które mają zostać ostylowane. Dzięki selektorom możemy decydować, do których części strony zostaną zastosowane konkretne style.

Najprostszym selektorem jest selektor znacznika, który odnosi się do wszystkich elementów danego typu, np. p, h1 czy div. Oznacza to, że wszystkie takie elementy na stronie otrzymają ten sam styl.

Selektor klasy (class) pozwala stylować wiele różnych elementów, które mają przypisaną tę samą klasę. W CSS zapisuje się go za pomocą kropki, np. .przycisk.

Selektor identyfikatora (id) służy do stylowania jednego, konkretnego elementu. W CSS zapisuje się go za pomocą znaku #, np. #naglowek. Id powinno być unikalne na stronie.

Można również łączyć selektory, aby precyzyjniej określić elementy, np. div p oznacza wszystkie paragrafy znajdujące się wewnątrz elementu div.

Istnieją także selektory grupowe, które pozwalają przypisać te same style do wielu elementów jednocześnie, np. h1, h2, p { ... }.

Selektory są podstawą CSS, ponieważ bez nich nie da się określić, które elementy mają być stylizowane.

<!-- HTML -->
<div class="sel-demo">

  <h1>Strona testowa CSS</h1>

  <h2 id="naglowek">Nagłówek z ID</h2>

  <p>Zwykły paragraf</p>

  <p class="tekst">Paragraf z klasą</p>
  <p class="tekst">Drugi paragraf z klasą</p>

  <div>
    <p>Paragraf w div (zagnieżdżenie)</p>
  </div>

  <h2>Drugi nagłówek</h2>

</div>

<!-- CSS -->
<style>
/* selektor znacznika */
.sel-demo p {
  color: #444;
}

/* selektor klasy */
.sel-demo .tekst {
  color: green;
  font-weight: bold;
}

/* selektor id */
.sel-demo #naglowek {
  color: blue;
  text-align: center;
}

/* selektor grupowy */
.sel-demo h1,
.sel-demo h2 {
  text-decoration: underline;
}

/* selektor zagnieżdżony */
.sel-demo div p {
  font-style: italic;
  background: #f0f6ff;
  padding: 4px;
}
</style>
