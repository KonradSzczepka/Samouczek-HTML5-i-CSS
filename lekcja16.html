Lekcja 16: Formularze

Wróć na główną!

Teoria

Formularze w HTML służą do zbierania danych od użytkownika, np. logowania, rejestracji, wysyłania wiadomości lub ankiet.

Podstawowym elementem formularza jest znacznik <form>, który zawiera wszystkie pola i przyciski. Formularz może wysyłać dane na serwer za pomocą atrybutu action oraz określać sposób wysyłania danych przez method (GET lub POST).

Do opisywania pól formularza używa się znacznika <label>, który jest powiązany z inputem poprzez atrybut for oraz id. Dzięki temu użytkownik może kliknąć w etykietę, aby aktywować pole.

Najważniejszym elementem formularza jest <input>, który może przyjmować różne typy określane przez atrybut type.

Najczęściej używane typy inputów to: text (tekst), password (hasło), email (adres e-mail), number (liczby), tel (telefon), date (data), checkbox (pole wyboru), radio (wybór jednej opcji), file (plik do przesłania) oraz submit (przycisk wysyłania formularza).

Input type="submit" tworzy przycisk, który wysyła formularz. Można też użyć przycisku <button>, ale submit jest najprostszą formą wysyłania danych.

Atrybut required wymusza wypełnienie pola, placeholder pokazuje podpowiedź w polu, a name określa nazwę pola wysyłaną na serwer.

Checkbox pozwala zaznaczać wiele opcji, natomiast radio pozwala wybrać tylko jedną opcję z grupy.

Formularze są podstawą interaktywnych stron internetowych i są używane w prawie każdej aplikacji webowej.

Przykład

<form>

  <label>Imię:</label>
  <input type="text" placeholder="Wpisz imię" required>

  <label>Email:</label>
  <input type="email">

  <label>Hasło:</label>
  <input type="password">

  <label>Wiek:</label>
  <input type="number">

  <label>Data urodzenia:</label>
  <input type="date">

  <!-- checkbox -->
  <input type="checkbox"> HTML
  <input type="checkbox"> CSS

  <!-- radio (jedna opcja) -->
  <input type="radio" name="level"> Beginner
  <input type="radio" name="level"> Pro

  <!-- przycisk wysyłania -->
  <input type="submit" value="Wyślij">

</form>
