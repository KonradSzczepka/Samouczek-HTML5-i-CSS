Lekcja 11: Tekst CSS

Wróć na główną!

Teoria

CSS pozwala na bardzo szerokie formatowanie tekstu, czyli zmianę jego wyglądu bez zmiany struktury HTML. Dzięki temu można dostosować wygląd strony do potrzeb użytkownika i projektu.

Jedną z podstawowych właściwości jest font-family, która określa czcionkę tekstu, np. Arial, Verdana lub Times New Roman. Można również podać kilka czcionek jako zapasowe.

Właściwość font-size odpowiada za rozmiar tekstu. Można ją ustawiać w pikselach (px), procentach (%) lub jednostkach względnych (em, rem).

Text-align służy do wyrównania tekstu. Może ustawiać tekst do lewej (left), prawej (right), środka (center) lub do pełnego wyjustowania (justify).

Text-decoration pozwala dodawać efekty do tekstu, takie jak podkreślenie (underline), przekreślenie (line-through) lub usunięcie dekoracji (none).

Font-weight określa grubość tekstu, np. normal lub bold, co pozwala wyróżniać ważne fragmenty treści.

Text-transform zmienia wielkość liter w tekście, np. na wielkie litery (uppercase), małe litery (lowercase) lub każde słowo z wielkiej litery (capitalize).

Line-height ustawia wysokość linii, co wpływa na czytelność tekstu poprzez zwiększenie odstępów między liniami.

Letter-spacing oraz word-spacing pozwalają kontrolować odstępy między literami i słowami, co może poprawić estetykę tekstu.

Wszystkie te właściwości razem pozwalają w pełni kontrolować wygląd tekstu na stronie internetowej.

Przykład

Formatowanie tekstu CSS

To jest zwykły tekst stylowany CSS. ważny fragment oraz tekst usunięty.

CSS pozwala zmieniać czcionkę, rozmiar tekstu, odstępy i wyrównanie. 

<!-- HTML -->

<h3 class="title-demo">Formatowanie tekstu CSS</h3>

<p class="text-demo">
  To jest zwykły tekst stylowany CSS.
  <span class="accent">ważny fragment</span>
  oraz <span class="strike">tekst usunięty</span>
</p>

<p class="text-demo">
  CSS pozwala zmieniać czcionkę, rozmiar tekstu, odstępy i wyrównanie.
</p>

<!-- CSS -->

<style>
.text-demo {
  font-family: Arial, sans-serif;
  font-size: 18px;
  color: #333;
  line-height: 1.6;
}

.title-demo {
  text-align: center;
  text-decoration: underline;
  font-size: 26px;
  font-weight: bold;
}

.accent {
  text-transform: uppercase;
  font-weight: bold;
  letter-spacing: 2px;
  color: #1a5fb4;
}

.strike {
  text-decoration: line-through;
}
</style>
