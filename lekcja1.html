Lekcja 1: Wprowadzenie do HTML

Wróć na główną!

Teoria

HTML (HyperText Markup Language) to język znaczników używany do tworzenia stron internetowych. Służy do opisywania struktury dokumentu, czyli tego jakie elementy znajdują się na stronie.

XML - HTML (HyperText Markup Language) to język znaczników używany do tworzenia stron internetowych. Służy do opisywania struktury dokumentu, czyli tego jakie elementy znajdują się na stronie.

Przeglądarka internetowa to program używany do wyświetlania stron internetowych. Służy do odczytywania kodu HTML i CSS oraz przedstawiania go w formie graficznej dla użytkownika.

Przykład

To jest przykładowy tekst.

<!DOCTYPE html>
<html lang="pl">
  <head>
    <meta charset="UTF-8">
    <title>Przykład</title>
  </head>
  <body>
    <p>To jest przykładowy tekst.</p>
  </body>
</html>
