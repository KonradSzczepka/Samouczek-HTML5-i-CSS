Lekcja 21: Pseudoelementy

Wróć na główną!

Teoria

Pseudoelementy w CSS pozwalają stylizować konkretne części elementu HTML, nawet jeśli nie istnieją one w kodzie jako osobne znaczniki.

Dzięki nim można np. dodać dekoracyjny tekst przed lub po elemencie, stylizować pierwszą literę lub pierwszą linię tekstu.

Najczęściej używane pseudoelementy to ::before i ::after, które wstawiają „wirtualne” elementy przed lub po zawartości.

Pseudoelementy są bardzo często używane do dekoracji UI, np. ikon, podkreśleń, animowanych efektów i ozdobników bez dodawania dodatkowego HTML.

Można je stylizować jak normalne elementy: nadawać im rozmiar, kolor, tło, a nawet animacje.

Tytuł sekcji

Pseudoelementy pozwalają stylizować fragmenty tekstu bez zmiany HTML. 

<style>

/* BEFORE i AFTER */
.badge {
  display: inline-block;
  padding: 10px 15px;
  background: #1a5fb4;
  color: white;
  border-radius: 6px;
}

.badge::before {
  content: "🔥 ";
}

.badge::after {
  content: " ✨";
}

/* PODKREŚLENIE */
.title {
  position: relative;
  display: inline-block;
}

.title::after {
  content: "";
  position: absolute;
  left: 0;
  bottom: -5px;
  width: 100%;
  height: 3px;
  background: red;
}

/* PIERWSZA LITERA */
.text::first-letter {
  font-size: 40px;
  color: #1a5fb4;
  font-weight: bold;
}

</style>

<div class="badge">Nowość</div>

<p class="title">Tytuł sekcji</p>
