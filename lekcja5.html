Lekcja 5: Listy

Wróć na główną!

Teoria

Listy w HTML służą do porządkowania i grupowania informacji w sposób czytelny dla użytkownika. Umożliwiają tworzenie zestawień, kroków instrukcji lub hierarchii danych.

W HTML wyróżniamy trzy podstawowe typy list. Lista nienumerowana <ul> (unordered list) wyświetla elementy w postaci punktów. Lista numerowana <ol> (ordered list) przedstawia elementy w kolejności za pomocą numerów. Każdy element listy definiuje się za pomocą znacznika <li>.

Listy mogą być zagnieżdżane, czyli jedna lista może znajdować się wewnątrz drugiej. Dzięki temu można tworzyć bardziej szczegółowe struktury, np. kategorie i podkategorie.

Znacznik <ul> stosuje się wtedy, gdy kolejność elementów nie ma znaczenia, natomiast <ol> używa się w sytuacjach, gdzie kolejność jest istotna, np. instrukcje krok po kroku.

Element <li> reprezentuje pojedynczy punkt listy i zawsze musi znajdować się wewnątrz <ul> lub <ol>.

Przykład

Lista zakupów (ul):

  • Chleb
  • Mleko
  • Masło

Kolejność kroków (ol):

  1. Włącz komputer
  2. Otwórz edytor kodu
  3. Napisz stronę HTML

Lista zagnieżdżona:

  • Owoce
    • Jabłka
    • Banan
  • Warzywa
<!-- LISTA NIEUPORZĄDKOWANA -->
<ul>
  <li>Chleb</li>
  <li>Mleko</li>
  <li>Masło</li>
</ul>

<!-- LISTA NUMEROWANA -->
<ol>
  <li>Włącz komputer</li>
  <li>Otwórz edytor kodu</li>
  <li>Napisz stronę HTML</li>
</ol>

<!-- LISTA ZAGNIEŻDŻONA -->
<ul>
  <li>Owoce
    <ul>
      <li>Jabłka</li>
      <li>Banan</li>
    </ul>
  </li>
  <li>Warzywa</li>
</ul>
