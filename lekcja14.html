Lekcja 14: Display i układ

Wróć na główną!

Teoria

Właściwość CSS display określa sposób wyświetlania elementu na stronie oraz to, jak zachowuje się on w układzie strony.

Jednym z najważniejszych typów jest display: block. Elementy blokowe zajmują całą szerokość dostępnego miejsca i zawsze zaczynają się w nowej linii. Przykładami są div, p oraz h1.

Drugim typem jest display: inline. Elementy inline zajmują tylko tyle miejsca, ile potrzebują i nie wymuszają nowej linii. Przykładami są span oraz a.

Istnieje również display: inline-block, który łączy cechy obu poprzednich. Element zachowuje się jak inline (ustawia się obok innych), ale można mu nadać szerokość i wysokość jak block.

W nowoczesnym CSS bardzo ważny jest display: flex. Flexbox pozwala łatwo ustawiać elementy w rzędzie lub kolumnie oraz kontrolować ich wyrównanie, odstępy i rozkład przestrzeni.

Flex działa na zasadzie kontenera i elementów wewnętrznych. Kontener otrzymuje display: flex, a jego dzieci automatycznie stają się elementami flex.

W flexboxie ważne są właściwości takie jak justify-content (wyrównanie w poziomie), align-items (wyrównanie w pionie) oraz gap (odstęp między elementami).

Wcześniej często używano float do układania elementów obok siebie, jednak obecnie jest to rozwiązanie przestarzałe i zastępowane przez flex.

Właściwość display jest jedną z najważniejszych w CSS, ponieważ decyduje o tym, jak elementy układają się na stronie.

Przykład

✔ BLOCK

Element block 1

Element block 2

✔ INLINE

Inline 1 Inline 2 Inline 3

✔ INLINE-BLOCK

A
B
C

✔ FLEX

Box 1
Box 2
Box 3
<!-- BLOCK -->
<p>Element block 1</p>
<p>Element block 2</p>

<!-- INLINE -->
<span>Inline 1</span>
<span>Inline 2</span>
<span>Inline 3</span>

<!-- INLINE-BLOCK -->
<div>A</div>
<div>B</div>
<div>C</div>

<!-- FLEX -->
<div class="flex-demo">
  <div>Box 1</div>
  <div>Box 2</div>
  <div>Box 3</div>
</div>

<style>
/* BLOCK */
.block-demo p {
  display: block;
}

/* INLINE */
.inline-demo span {
  display: inline;
}

/* INLINE-BLOCK */
.inline-block-demo div {
  display: inline-block;
  width: 120px;
  height: 60px;
}

/* FLEX */
.flex-demo {
  display: flex;
  justify-content: space-between;
  gap: 10px;
}
</style>
